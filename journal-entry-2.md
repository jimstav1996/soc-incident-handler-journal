## 📄 Description
This journal entry documents an unauthorized access investigation involving multiple failed login attempts followed by a successful privileged account login. This maps to the Detection and Analysis phase of the NIST Incident Response Lifecycle because the focus was to identify attacker access methods, affected systems, and initial scope.

## ❓ The 5 W's
- **Who:** Unknown threat actor targeting privileged access
- **What:** Multiple failed SSH login attempts followed by a successful root login
- **When:** Early morning during log review
- **Where:** Linux authentication server
- **Why:** Attempt to gain unauthorized privileged access for persistence or lateral movement

## 🧠 Notes
